# 矩阵的乘法 #

- 矩阵的线性运算：矩阵加减、数乘矩阵（上一次讲的）

- 为矩阵乘法做准备

- 可交换矩阵
- 线性方程组的矩阵表示
- 线性变换的矩阵表示
