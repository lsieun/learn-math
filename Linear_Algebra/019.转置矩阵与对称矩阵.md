# 转置矩阵与对称矩阵 #

矩阵的运算包括：

- 矩阵的线性运算
- 矩阵的乘法
- 矩阵的转置

矩阵的乘法，目前包括：**矩阵的幂**、**旋转变换**。

这一节包括3个部分：

- 转置矩阵：转置矩阵的定义、矩阵转置的运算性质
- 对称矩阵：对称矩阵是矩阵转置的一种特殊形式，而且该矩阵必须是n阶方阵
- 反对称阵：反对称矩阵是对称矩阵的一种变形
