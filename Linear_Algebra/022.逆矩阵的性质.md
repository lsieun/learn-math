# 矩阵的性质 #

主要有两部分：

- 逆矩阵的运算律
- 伴随矩阵的性质

逆矩阵的运算律有以下内容：

- （1）矩阵自身求逆
- （2）数乘-->求逆
- （3）矩阵相乘-->求逆
- （4）转置-->求逆
- （5）求逆-->行列式

伴随矩阵有以下内容：

- （1）伴随矩阵的行列式
- （2）伴随矩阵 用 原矩阵 和 行列式表示
